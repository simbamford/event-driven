Description

Controls the iCub to verge on a target

Parameters

  • – name : Specifies the stem name of ports created by the module.

Input Ports

  • /vVergence/vBottle:i [eventdriven::vBottle] [default carrier:udp]: Accepts the address events in the vBottle container

Output Ports

  • /vVergence/debug:o [yarp::sig::ImageOf] [default carrier:tcp]: Outputs a debugging image displaying events from left and right cameras
  • /vVergence/scopefilters:o [yarp::os::Bottle] [default carrier:udp]: Outputs debug information for use with yarpscope. Energies of several filters are shown with different colors.
